# 🚀 Quick Start - Animations & Languages

## ⚡ البداية السريعة

### 1️⃣ افتح الموقع
```bash
# افتح index.html في المتصفح
```

### 2️⃣ شاهد الانيميشنز
- الصفحة هتفتح بانيميشن سلس
- الـ Navbar هينزل من فوق
- المنتجات هتظهر واحدة ورا التانية

### 3️⃣ جرب تبديل اللغة
- دور على زر اللغة (🌐) في الـ Navbar
- اضغط عليه
- الموقع هيتحول للعربية تلقائياً

### 4️⃣ جرب الانيميشنز التفاعلية
- مرر الماوس على أي منتج → هيطلع لفوق مع shadow
- اسكرول لتحت → العناصر هتظهر تدريجياً
- اضغط على أي زر → هتشوف ripple effect

## 🎨 صفحة التجربة

افتح `animation-demo.html` لرؤية:
- ✅ جميع أنواع الانيميشنز
- ✅ أمثلة تفاعلية
- ✅ أزرار للتحكم

## 🔥 نصائح سريعة

1. **للحصول على أفضل تجربة:**
   - استخدم Chrome أو Firefox
   - فعّل JavaScript
   - اتصل بالإنترنت (لتحميل GSAP)

2. **لتجربة الانيميشنز:**
   - افتح animation-demo.html
   - اضغط على الأزرار
   - اسكرول لأسفل

3. **لتبديل اللغة:**
   - اضغط زر اللغة في Navbar
   - أو افتح Console واكتب:
   ```javascript
   setLanguage('ar') // للعربية
   setLanguage('en') // للإنجليزية
   ```
