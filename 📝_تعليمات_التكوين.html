📧 تعليمات تكوين EmailJS

⚠️ مهم جداً: يجب الحصول على 3 مفاتيح من EmailJS Dashboard

1 Public Key (المفتاح العام)

📍 من أين تحصل عليه:

  • اذهب إلى: https://dashboard.emailjs.com/
  • Account → General
  • انسخ Public Key
📋 مثال:
user_K7xYz9AbCdEfGhIjKlMnOp
[ضع Public Key هنا]

📝 أين تضعه في الكود:

// في ملف contact.html (حوالي السطر 255) const PUBLIC_KEY = "ضع_مفتاحك_هنا";

2 Service ID (معرف الخدمة)

📍 من أين تحصل عليه:

  • EmailJS Dashboard → Email Services
  • اختر Gmail Service
  • انسخ Service ID
📋 مثال:
service_abc1234
[ضع Service ID هنا]

📝 أين تضعه في الكود:

// في ملف contact.html (حوالي السطر 268) window.EMAILJS_SERVICE_ID = "ضع_Service_ID_هنا";

3 Template ID (معرف القالب)

📍 من أين تحصل عليه:

  • EmailJS Dashboard → Email Templates
  • اختر Template أو أنشئ واحد جديد
  • انسخ Template ID
📋 مثال:
template_xyz7890
[ضع Template ID هنا]

📝 أين تضعه في الكود:

// في ملف contact.html (حوالي السطر 269) window.EMAILJS_TEMPLATE_ID = "ضع_Template_ID_هنا";

✅ مثال كامل بعد التعديل

<script> // ✅ Public Key من EmailJS Dashboard const PUBLIC_KEY = "user_K7xYz9AbCdEfGhIj"; if (PUBLIC_KEY === "YOUR_PUBLIC_KEY" || PUBLIC_KEY === "") { setTimeout(() => { alert('⚠️ EmailJS غير مكون!'); }, 1000); } else { emailjs.init(PUBLIC_KEY); } // ✅ Service ID و Template ID window.EMAILJS_SERVICE_ID = "service_gmail123"; window.EMAILJS_TEMPLATE_ID = "template_contact456"; </script>
