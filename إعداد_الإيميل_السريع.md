## ⚡ خطوات سريعة (5 دقائق)

### 1️⃣ إنشاء حساب EmailJS
```
1. روح https://www.emailjs.com/
2. اضغط Sign Up
3. سجل بإيميلك
4. تأكد من الإيميل
```

### 2️⃣ ربط Gmail
```
1. Email Services → Add New Service
2. اختر Gmail
3. Connect Account
4. سجل دخول بـ abdelrhmansherif140@gmail.com
5. احفظ Service ID (مثل: service_abc123)
```

### 3️⃣ إنشاء Template
```
1. Email Templates → Create New Template
2. Subject: New Contact Form Message: {{subject}}
3. Body:
   From: {{from_name}}
   Email: {{from_email}}
   Message: {{message}}
4. To Email: abdelrhmansherif140@gmail.com
5. احفظ Template ID (مثل: template_xyz789)
```

### 4️⃣ نسخ Public Key
```
1. Account → General
2. انسخ Public Key (مثل: user_AbCdEfGhIjKlMnOp)
```

### 5️⃣ تحديث الكود

#### في `contact.html` (السطر 107):
```html
<script>
    (function(){
        emailjs.init("user_AbCdEfGhIjKlMnOp"); // ضع Public Key هنا
    })();
</script>
```

#### في `js/contact.js` (السطر 90):
```javascript
const response = await emailjs.send(
    'service_abc123',      // ضع Service ID هنا

```
