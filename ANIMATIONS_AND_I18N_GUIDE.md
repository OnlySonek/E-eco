### 🌍 نظام اللغات - Language System

تم إضافة دعم كامل للغتين العربية والإنجليزية:

#### المميزات:

- ✅ تبديل سلس بين العربية والإنجليزية
- ✅ دعم RTL للعربية
- ✅ حفظ اللغة المختارة في localStorage
- ✅ ترجمة تلقائية لجميع العناصر
- ✅ زر تبديل اللغة في الـ Navbar

#### كيفية الاستخدام:

1. **تبديل اللغة:**
   - اضغط على زر اللغة (🌐) في الـ Navbar
   - سيتم تحديث الصفحة تلقائياً

2. **إضافة ترجمات جديدة:**
   ```javascript
   // في ملف js/i18n.js
   const translations = {
       en: {
           'key.name': 'English Text'
       },
       ar: {
           'key.name': 'النص العربي'
       }
   };
   ```

3. **استخدام الترجمة في HTML:**
   ```html
   <element data-i18n="key.name">Default Text</element>
   ```

4. **استخدام الترجمة في JavaScript:**
   ```javascript
   import { t } from './i18n.js';
   const text = t('key.name');
   ```

## 🎨 تخصيص الانيميشنز

يمكنك تعديل الانيميشنز في `js/animations.js`:

```javascript
// مثال: تغيير مدة الانيميشن
gsap.from('.navbar-glass', {
    y: -100,
    opacity: 0,
    duration: 1.5, // غير هذا الرقم
    ease: 'power3.out'
});
```
