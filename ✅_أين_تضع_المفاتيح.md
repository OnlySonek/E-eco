## 🔑 المفاتيح الثلاثة المطلوبة

### 1️⃣ Public Key
```
📍 من: EmailJS Dashboard → Account → General
📋 مثال: user_K7xYz9AbCdEfGhIj
```

### 2️⃣ Service ID
```
📍 من: EmailJS Dashboard → Email Services
📋 مثال: service_gmail123
```

### 3️⃣ Template ID
```
📍 من: EmailJS Dashboard → Email Templates
📋 مثال: template_contact456
```

## 📝 المكان بالظبط في contact.html

### ابحث عن هذا الجزء (حوالي السطر 248):

```html
<!-- EmailJS SDK -->
<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script>
<script>
    // ============================================
    // 📧 EmailJS Configuration
    // ============================================
    
    // ✅ Step 1: Initialize EmailJS with Public Key
    const PUBLIC_KEY = "YOUR_PUBLIC_KEY"; // ⚠️ استبدل هنا
    
    if (PUBLIC_KEY === "YOUR_PUBLIC_KEY" || PUBLIC_KEY === "") {
        setTimeout(() => {
            alert('⚠️ EmailJS غير مكون!');
        }, 1000);
    } else {
        emailjs.init(PUBLIC_KEY);
    }
    
    // ✅ Step 2: Set Service ID and Template ID
    window.EMAILJS_SERVICE_ID = "YOUR_SERVICE_ID";    // ⚠️ استبدل هنا
    window.EMAILJS_TEMPLATE_ID = "YOUR_TEMPLATE_ID";  // ⚠️ استبدل هنا
</script>
```

## 🎯 خطوات التطبيق

### 1. احصل على المفاتيح من EmailJS:
```
□ Public Key من Account → General
□ Service ID من Email Services
□ Template ID من Email Templates
```

### 2. افتح contact.html:
```
□ ابحث عن السطر 255 تقريباً
□ ابحث عن "EmailJS Configuration"
```

### 3. استبدل المفاتيح:
```
□ استبدل YOUR_PUBLIC_KEY بمفتاحك
□ استبدل YOUR_SERVICE_ID بـ Service ID
□ استبدل YOUR_TEMPLATE_ID بـ Template ID
```

### 4. احفظ الملف:
```
□ Ctrl+S أو File → Save
```

### 5. اختبر:
```
□ افتح contact.html في المتصفح
□ املأ الفورم
□ اضغط Send
□ انتظر Alert النجاح
□ تحقق من Gmail
```

## 📊 مثال كامل

### الكود الكامل بعد التعديل:

```html
<!-- EmailJS SDK -->
<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script>
<script>
    // ============================================
    // 📧 EmailJS Configuration
    // ============================================
    
    // ✅ Step 1: Initialize EmailJS with Public Key
    // Get your Public Key from: https://dashboard.emailjs.com/admin/account
    const PUBLIC_KEY = "user_K7xYz9AbCdEfGhIj"; // ✅ تم التعديل
    
    if (PUBLIC_KEY === "YOUR_PUBLIC_KEY" || PUBLIC_KEY === "") {
        // تحذير للمطور فقط
        setTimeout(() => {
            alert('⚠️ تحذير للمطور:\n\nEmailJS غير مكون!\n\nيجب تحديث:\n1. Public Key\n2. Service ID\n3. Template ID\n\nفي ملف contact.html\n\nراجع: 🔧_حل_مشكلة_الإيميل.md');
        }, 1000);
    } else {
        emailjs.init(PUBLIC_KEY);
    }
    
    // ✅ Step 2: Set Service ID and Template ID
    // Get these from: https://dashboard.emailjs.com/admin
    window.EMAILJS_SERVICE_ID = "service_gmail123";      // ✅ تم التعديل
    window.EMAILJS_TEMPLATE_ID = "template_contact456";  // ✅ تم التعديل
</script>

<script type="module" src="js/main.js"></script>
<script type="module" src="js/contact.js"></script>
</body>
</html>
```
