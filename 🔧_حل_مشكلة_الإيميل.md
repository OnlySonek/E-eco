## 🎯 الحل السريع

### في `contact.html` (حوالي السطر 107):

#### ❌ قبل:
```html
<script>
    const PUBLIC_KEY = "YOUR_PUBLIC_KEY";
    emailjs.init(PUBLIC_KEY);
    
    window.EMAILJS_SERVICE_ID = "YOUR_SERVICE_ID";
    window.EMAILJS_TEMPLATE_ID = "YOUR_TEMPLATE_ID";
</script>
```

#### ✅ بعد:
```html
<script>
    const PUBLIC_KEY = "user_K7xYz9AbCdEfGhIj"; // ضع مفتاحك هنا
    emailjs.init(PUBLIC_KEY);
    
    window.EMAILJS_SERVICE_ID = "service_gmail123"; // ضع Service ID هنا
    window.EMAILJS_TEMPLATE_ID = "template_contact456"; // ضع Template ID هنا
</script>
```

## 📋 من أين تحصل على المفاتيح

### 1. Public Key:
```
EmailJS Dashboard → Account → General
انسخ: Public Key
```

### 2. Service ID:
```
EmailJS Dashboard → Email Services
انسخ: Service ID (مثل: service_abc123)
```

### 3. Template ID:
```
EmailJS Dashboard → Email Templates
انسخ: Template ID (مثل: template_xyz789)
```

## 🐛 أخطاء شائعة

### 1. "EmailJS is NOT loaded"
```
الحل: تأكد إن السطر ده موجود:
<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script>
```

### 2. "Public Key not configured"
```
الحل: استبدل YOUR_PUBLIC_KEY بالمفتاح الحقيقي
```

### 3. "Service ID not configured"
```
الحل: استبدل YOUR_SERVICE_ID بالـ ID الحقيقي
```

### 4. "Template ID not configured"
```
الحل: استبدل YOUR_TEMPLATE_ID بالـ ID الحقيقي
```

## ✅ Checklist

- [ ] فتحت Console (F12)
- [ ] شفت رسائل التكوين
- [ ] لا توجد رسائل حمراء (❌)
- [ ] Public Key ليس "YOUR_PUBLIC_KEY"
- [ ] Service ID ليس "YOUR_SERVICE_ID"
- [ ] Template ID ليس "YOUR_TEMPLATE_ID"
- [ ] جربت إرسال الفورم
- [ ] شفت Alert (نجاح أو فشل)
- [ ] قرأت Error Message في Console
